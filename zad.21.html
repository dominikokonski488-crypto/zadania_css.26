Co znaczy skrót ID? ID znaczy Identyfikator.
Jaki jest cel stosowania ID? ID="nazwa" Przypisuje nazwę elementowi (identyfikator), która nie może się powtarzać w całym dokumencie
Jakie jest znaczenie znaku # Znak # w CSS oznacza odwołanie do identyfikatora ID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie następuje przez zapis w CSS: #nazwaID oraz w HTML: id="nazwaID".
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. ID musi być unikalne na stronie.
  2. Jeden element może mieć tylko jedno ID.
  3. ID często używa się do stylowania lub odwołań w CSS i JavaScript.
