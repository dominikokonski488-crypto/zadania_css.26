Uwagi na temat pisania kodu w CSS:
1. Właściwości CSS zapisujemy jako pary: cecha: wartość, oddzielone średnikiem.
2. Blok deklaracji umieszczamy w nawiasach klamrowych { }.

Definicja CSS – Kaskadowych Arkuszy Stylów:
CSS (Cascading Style Sheets) to język służący do opisu sposobu prezentacji dokumentów HTML. Umożliwia oddzielenie treści strony od jej wyglądu, co ułatwia zarządzanie i modyfikację wielu stron jednocześnie.

Definition of CSS – Cascading Style Sheets:
CSS (Cascading Style Sheets) is a language used to describe the presentation of HTML documents. It allows the separation of content from design, making it easier to manage and modify multiple pages at once.
