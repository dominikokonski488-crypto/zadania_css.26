1) Co to są pseudoklasy?

CSS przewiduje kilka ściśle określonych klas pozwalających na formatowanie niektórych elementów dokumentu – są to pseudoklasy. Pozwalają one stylizować elementy w zależności od ich stanu lub położenia w dokumencie.

2) Uwaga dotycząca dwukropka

Szalenie istotnym elementem w konstrukcji pseudoklasy jest dwukropek występujący pomiędzy selektorem a nazwą klasy, np. A:hover.

3) Tabela opisująca pseudoklasy

Pseudoklasa Opis
A:link Odpowiada za wygląd hiperłącza, które nie zostało jeszcze odwiedzone.
A:visited Odpowiada za formatowanie hiperłącza już odwiedzonego.
A:hover Zmiany wyglądu elementu po najechaniu na niego kursorem myszy.
A:active Formatowanie elementu po wciśnięciu przycisku myszy, ale przed jego zwolnieniem (link aktywny/klikany).
A:focus Formatowanie elementu, który posiada fokus (zaznaczony klawiszem Tab).
:lang Formatowanie wielojęzykowych dokumentów – dopasowuje elementy w określonym języku.
:first-child Umożliwia określenie formatowania dla pierwszego dziecka wybranego elementu.

4) Zastosowanie pseudoklasy :active

Pseudoklasa :active może być używana:
1) Jako uzupełnienie definicji odnośników (link wciśnięty).
2) Jako element poprawiający czytelność formularzy.

5) Trzy przykłady kodów definiowania pseudoklas

Przykład 1: :link i :visited
Temat: Formatowanie odnośnika przed i po odwiedzeniu.

A:link    { color: red; }
A:visited { color: green; }

Przykład 2: Pełne formatowanie odnośników
Temat: Formatowanie wyglądu wszystkich odnośników w dokumencie.

A { font-size: 12pt; text-decoration: none; }
A:link    { color: navy; }
A:visited { color: green; }
A:hover   { color: red; text-decoration: underline; }

Przykład 3: :focus
Temat: Zastosowanie pseudoklasy :focus.

A:focus { color: brown; }
